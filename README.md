# Key-Values Splitter
A function that receives an input dictionary and returns a nested array of keys and values.


## Requirements ⚙️
- Node.js
- Typescript

## Run 🏁
```bash
C:\project> tsc main.ts
C:\project>  node main.js
[ [ 'a', 'b', 'c' ], [ 1, 2, 3 ] ]
[ [ 'a', 'b', 'c' ], [ 'Apple', 'Microsoft', 'Google' ] ]
[ [ 'key1', 'key2', 'key3' ], [ true, false, undefined ] ]
```
